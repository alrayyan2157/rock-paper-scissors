# Rock Paper Scissors Game

A classic Rock Paper Scissors game implemented in JavaScript that runs in your browser console. Play against the computer in a best-of-5 rounds match!

## 🎮 Game Rules

- **Rock** beats Scissors
- **Paper** beats Rock
- **Scissors** beats Paper
- If both players choose the same option, it's a draw

The game consists of 5 rounds. The player with the most wins at the end is declared the winner!

## 🚀 How to Play

1. **Open the game**: Open `index.html` in your web browser
2. **Open Developer Console**:
   - **Chrome/Edge**: Press `F12` or `Ctrl+Shift+I` (Windows/Linux) / `Cmd+Option+I` (Mac)
   - **Firefox**: Press `F12` or `Ctrl+Shift+K` (Windows/Linux) / `Cmd+Option+K` (Mac)
   - **Safari**: Enable Developer menu in Preferences > Advanced, then press `Cmd+Option+C`
3. **Start playing**: The game will automatically prompt you for your choices
4. **Enter your choice**: Type "rock", "paper", or "scissors" when prompted
5. **View results**: Check the console for round results and final score

## 🎯 Features

- Interactive console-based gameplay
- Score tracking throughout the 5-round match
- Clear win/loss/draw announcements for each round
- Final game result declaration
- Input validation for invalid choices
